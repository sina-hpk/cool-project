# Network Swiss Army Knife 🛠️

A multifunctional Python tool for network sniffing, ARP spoofing, MITM forwarding, and DNS spoofing.

## ⚡ Features

- Passive Sniffer
- Network ARP Scan
- Save to CSV
- ARP Spoofing
- MITM Packet Forwarder
- DNS Spoofing

---

## 🚀 Installation

pip install -r requirements.txt

---

## 💻 Usage

python snifferv2.py
